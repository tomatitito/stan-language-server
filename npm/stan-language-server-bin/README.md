# stan-language-server-bin

Standalone CLI distribution of the [Stan language server](https://github.com/tomatitito/stan-language-server).

This package provides the `stan-language-server` binary as a single bundled JavaScript file with no additional dependencies. It requires Node.js (>=18) to run.

## Installation

```bash
npm install -g stan-language-server-bin
```

## Usage

```bash
stan-language-server --stdio
```

See the [main repository](https://github.com/tomatitito/stan-language-server) for more detailed usage information.
